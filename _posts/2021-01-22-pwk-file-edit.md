## VIM

```bash
dd → delete current line
yy → copy current line
P → paste line
x → delete letter
```

## compare

```bash
comm file1 file2
diff -c file1 file2
diff -u file1 file2
vimdiff file1 file2
    → (ctrl + w) - switch windows
    → (D + O) - takes the difference and paste in current window
    → (D + P) - takes the difference and paste in other window
```
